# Todo Tasks Frontend V2

This is a frontend project for managing todo tasks, built with [Next.js](https://nextjs.org). It provides a modern and responsive user interface to create, update, and organize your tasks efficiently.

## Features

- User authentication and authorization
- Task creation, editing, and deletion
- Task categorization and filtering
- Responsive design for desktop and mobile
- Integration with backend APIs for data persistence

## Technologies Used

- [Next.js](https://nextjs.org) - React framework for server-side rendering and static site generation
- TypeScript - Typed JavaScript for improved developer experience and code quality
- Tailwind CSS - Utility-first CSS framework for styling
- React Query - Data fetching and state management
- Vercel - Deployment platform

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Run the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the app.
